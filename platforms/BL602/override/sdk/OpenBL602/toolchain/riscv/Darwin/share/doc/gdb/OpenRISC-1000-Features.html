G.5.10 Openrisc 1000 Features

The ‘org.gnu.gdb.or1k.group0’ feature is required for OpenRISC 1000 targets. It should contain the 32 general purpose registers (‘r0’ through ‘r31’), ‘ppc’, ‘npc’ and ‘sr’.
